Cologne Sanskrit Lexicon — four design-direction prototypes

Four visually and structurally distinct end-to-end explorations of the Cologne Digital Sanskrit Dictionaries redesign. Each direction is a complete static app over the same four pages — homepage/catalogue, search results, entry reading page, dictionary detail — and embeds the same real public-domain dictionary text (Monier-Williams 1899 · Apte 1890, from the Cologne csl-orig source): agní, dhárma, mánas, índra.

Grid: 4 directions × 4 pages = 16 screens. The single-screen v1 reference prototype is here; product rulings live in proposal-brief.md and ui-spec-app-v1.md.

D1 · Proposal A (ruled)

Research Workbench

agní Georgia serif · dense two-pane · university library

The ruled direction: one scholarly search surface, results list beside an always-open entry reader. Restrained blues and paper whites, everything one click deep.

All 17 pages are self-contained static HTML (inline CSS/JS, no build step, no external assets) served from GitHub Pages; the live Cologne server is not required.
